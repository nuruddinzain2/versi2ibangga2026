Panduan Penggunaan Dashboard

Dashboard ini menampilkan progress pendataan Sensus Ekonomi 2026 untuk seluruh PML dan PPL di wilayah Kota Banjarmasin. Data diambil langsung dari Google Sheets secara real-time saat halaman dibuka.

Cara menggunakan:

  • Masukkan NIP Task Force Anda pada kolom di bawah, lalu klik Cari.
  • Dashboard akan menampilkan ringkasan total Task Force Anda (Sandbox) beserta daftar PML dan PPL secara hierarkis.
  • Gunakan dropdown PPL untuk melihat detail satu PPL tertentu, termasuk ID SUB SLS dan Keterangan GC.
  • NIP Anda dapat disimpan di perangkat ini agar tidak perlu mengetik ulang di kemudian hari.
Masuk dengan NIP Task Force
Pengingat Privasi: Jika Anda mencentang "Ingat NIP saya", NIP akan disimpan di memori lokal perangkat (browser/HP) Anda. Data hanya tersimpan di perangkat ini dan bukan di server. Anda dapat menghapusnya kapan saja dengan menekan tombol Reset.
